# Analyst Onboarding Path

## Purpose

Define a practical learning path for new analysts joining the finance operations process.

## Path

1. Understand receipts and payment batches.
2. Learn invoice and reservation references.
3. Review allocation-ready, evidence-review, missing-evidence, cancellation-fee, amount-variance, and rejected-card statuses.
4. Practice reviewing open balances by aging bucket.
5. Review chargeback and rejected-card examples.
6. Learn how to document decisions in the knowledge library.

## Completion Criteria

The analyst can explain why an item is allocatable, open, or routed to exception handling.
