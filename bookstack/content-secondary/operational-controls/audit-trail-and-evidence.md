# Audit Trail and Evidence

## Purpose

Describe the evidence required to support reconciliation decisions.

## Evidence Types

- payment receipt reference
- payment batch identifier
- invoice reference
- reservation reference
- payment-channel token mapping
- exception classification
- reviewer note
- approval record

## Control Principle

Every resolved item should be traceable from operational decision back to source evidence.

If the evidence is incomplete, the item remains open and should not be treated as fully allocated.
