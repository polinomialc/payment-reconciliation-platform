# Daily Control Checklist

## Purpose

Provide a repeatable daily control checklist for finance operations.

## Checklist

1. Confirm receipt files were received for the expected payment channels.
2. Confirm payment batch exports were loaded.
3. Check whether any source file is missing, duplicated, or unusually small.
4. Run reconciliation and verify output counts by status.
5. Review rejected card transactions, over/under payments, and cancellation-fee cases.
6. Review open balances by aging bucket.
7. Document unresolved items and assigned owners.

## Expected Evidence

- source file inventory
- reconciliation summary
- exception queue
- open-balance aging output
- reviewer notes
